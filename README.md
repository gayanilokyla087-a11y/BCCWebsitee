BCC Web Portal

This is my school project for Binalbagan Catholic College (BCC).
The website was created to practice and improve my skills in web development, particularly in designing responsive and user-friendly webpages using HTML, CSS, PHP, and MySQL.

The goal of this project is to present important school information in a clean, organized, and professional layout while following the required design and color theme.

📄 Pages Included
🏠 Home Page

The homepage contains:

A navigation bar for easy access to different pages
A hero section with a welcome message and school image
A short introduction about BCC
A “Learn More” button
A courses preview section
A contact information and message form section
📘 About Page

The About page uses a side-by-side layout to present:

The history of Binalbagan Catholic College
The school’s Vision and Mission
Information about the institution and its goals

The page is designed to provide visitors with a better understanding of the school.

📚 Courses Page

The Courses page displays different academic departments using modern course cards.

Courses included:

BS Information Technology
Business Administration
Education

Each card contains an icon, course title, and short description.

👨‍🏫 Faculty Page

The Faculty page presents teachers and faculty members using organized profile cards to maintain a clean and professional appearance.

🎨 Design and Colors

The website follows the official color palette required in the project instructions:

Navy Blue (#001f3f)
Used for headings, buttons, navigation bar, and important sections.
White (#ffffff)
Used for the background and content cards to keep the layout clean and readable.
Light Gray (#f4f4f4)
Used to separate sections and improve visual balance.

The design focuses on simplicity, readability, and proper spacing to create a professional-looking school website.
